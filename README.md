# Sales-Analysis-Dashboard
An interactive Sales Analysis Dashboard created using Microsoft Excel to analyze sales performance, customer behavior, product revenue, profit trends, and regional insights. 
This dashboard helps businesses make data-driven decisions through clear visualizations and KPIs.

## Features
-  Monthly Revenue Analysis
- Highest Revenue Product Analysis
- Highest Profit Product Tracking
- Customer Segment Distribution
- Region-wise Revenue Analysis
- State-wise Revenue Comparison
- Order Status Overview
- Quarterly Revenue Trends
- Interactive Slicers & Filters

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting
- Dashboard Designing

## Key KPIs

 Total Revenue | ₹7.38 M 
 Total Profit | ₹275,288 
 Total Quantity | 378 
 Average Order Value (AOV) | 1,476,450 

## Dataset Used
The dashboard is built using the following datasets:
- Orders Data
- Sales Data
- Product Data
- Customer Data

## Insights Generated
- Laptop category generated the highest revenue.
- Monitor category produced the highest profit.
- Youth customers contributed the highest revenue share.
- South region generated maximum revenue.
- Home Office segment showed strong customer engagement.
